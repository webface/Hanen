WordPress plugin scaffold which makes use of:

1) WordPress Boilerplate Plugin: http://wppb.io/
2) Harri Bell-Thomas Page Templates http://www.wpexplorer.com/wordpress-page-templates-plugin/
3) WPHuman's WordPress Settings Module: https://github.com/wphuman/WordPress-Settings-Module

== Installation ==

1. Download the plugin
2. Upload plugin directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
